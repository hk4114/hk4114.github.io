```
REPO
   config
      config.ts #配置入口 会包含路由配置和构建配置
      routes.ts #路由配置
      buildConf.ts #构建配置 可选
   src
   test
   types #全局类型定义文件
   .editorconfig 编辑器统一配置
   .pretterrc.js 代码风格格式化配置
   tsconfig.json ts 配置文件

src/
   common 业务公共处理集和
      index.ts 统一对外暴露
   components 独立组件目录 不含业务
      index.ts
   const       通用常量
      index.ts
   entity-types 统一index导出
      UserEntitu.ts
      index.ts
   layout 布局 一个项目会有多个页面
      MainLayout.ts
      MainLayout.less
   less
      var.less 常用变量汇总
      common.less xi项目级别
   locale 多语言
      zh_CN.ts
   model 视图模型(vm)目录
      user.ts
   page
   services api
      ServiceBase.ts 一个项目一个base服务 处理公共逻辑
      xxx.service.ts
   utils 工具方法目录 统一导出 复用程度更高
      index.ts
   global.less 默认文件无效改动
```
