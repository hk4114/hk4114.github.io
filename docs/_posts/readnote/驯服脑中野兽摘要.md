## 饮食

### 原则：
- 多吃对大脑有益的食物。
- 少吃对大脑有害的食物。
- 不限制卡路里。
没有必要减少饭量，吃饱也没关系。
少吃对大脑有害的食物就可以了，不需要完全不吃。

对大脑有益的食物
|   类别   |             例             |                  推荐摄入量                  |    用手测量的分量    |
| :------: | :------------------------: | :------------------------------------------: | :------------------: |
|  全谷物  |     糙米、燕麦、藜麦等     |     一周吃21份(一日三餐各1份，1份=125g)      |  约一个拳头大小的量  |

|  坚果类  |  核桃、夏威夷果、巴旦木等  |              一天吃1份(1份=20g)              |   约为拇指大小的量   |

### 细节/方法论
1. 咖啡配合绿茶饮用，咖啡不宜多喝，建议每天1罐(400mg咖啡因)，起床后90min内不喝
2. 地中海饮食(多吃蔬菜、水果和鱼，少吃其他动物制品) todo: 关于地中海饮食展开说明

::: tip
示例：

早餐      钢切燕麦粥加蓝莓和巴旦木/意式菠菜甘蓝蘑菇蛋饼
午餐      糙米/炒鸡肉、西红柿、大豆和薯类/沙拉（用甘蓝、藜麦、巴旦木、西红柿、西蓝花制作并加入橄榄油和苹果醋）
晚餐      撒有碎核桃的烤三文鱼/鸡胸肉、西蓝花、腰果沙拉
:::

### 检验效果
专注力日志+记分板
请每隔一个小时就思考一下“自己专注多久了”，以10分为满分给自己打分。
根据主观判断给自己打分就可以，“完全不在意周围发生的事情，全神贯注地工作”是10分，“几乎没能着手工作”是0分，如果觉得“专注度和平时一样”，就给自己打5分。
像这样，写出你的专注力日志。用Excel将记录下的数据做成了折线图。

### 基本设定表

#### 具象想象
1. 不要用专业术语。不要说“建设可持续发展的社会”，而要用“建设混合动力汽车变多的社会”这样的表达。
2. 不要用数字。不要说“一年瘦10公斤”，而要用“能穿上20多岁时买的牛仔裤”这样的表达，后者的效果更好。当然，数字对我们了解自己的完成度来说很重要，但在这一阶段，还是要更加重视目标的具象性。

::: tip 案例
> case 完成企划书　

把企划书交给上司以后松了一口气，并去看了电影

> case每天跑步　

每天跑步增强体魄，总是头脑清醒地完成工作
:::

#### 反向规划
设定在完成最终目标前需要完成的子目标和完成子目标的日期。

需要要注意的是，不是以通常的从现在到未来的方式设定目标，而是从最终目标开始**倒推**，设定短期目标。

::: tip 案例
具象目标: 把企划书交给上司以后松了一口气

反向规划: 在交企划书的前一天把企划书再看一遍→在交企划书的三天前完成企划书→在交企划书的五天前提出解决方案→在交企划书的七天前完成调查
　
目标具象: 每天跑步增强体魄

反向规划: 3个月后累计跑100千米→1个月后累计跑25千米→14天后累计跑10千米
:::

#### 每日任务设定
把目标进行细化是任务管理的基本原则。对将目标细化到什么程度没有固定的标准，以**能在几分钟到一个小时内完成这个目标**为宜。

::: tip 案例
子目标: 在截止日期七天前完成调查和分析

每日任务: 对专业人士进行访谈，从网上收集必要的资料，阅读收集的资料等

子目标: 14天后累计跑10千米

每日任务: 在跑步机上跑1千米，沿着平时的跑步路线跑2千米等
:::

基本设定表 并不是填一次就可以了，我们需要根据项目的情况定期修改。

其中修改最多的是“每日任务设定”。

很多人不能集中注意力是因为在这一步没有合理地细化任务。

当出现这种情况的时候，先回到“每日任务设定”这一步，对任务进行再一次细化。

如果“写企划概要”的任务不能让你集中注意力，就把它分解成2～3个子任务，如“从资料中提取有用的信息”和“整理提取的信息”。

### 实践设定表

#### 每日任务选择
从基本设定表中的每日任务中，选出3～5个“当天必须着手做的事情”或“最长也只需要2～3小时就能完成的事情”。

驯兽师无法同时处理3个以上的信息。总之，一天最多给自己设定5个任务，如果还有多余的时间再追加。

#### 对比目标和障碍  
写出在完成每日任务时可能会发生的问题。

::: tip 案例
如果每日任务是“在跑步机上跑1千米”，至少写出一个会妨碍你完成此任务的因素，如 工作太累，不想跑了,想去看电视等。
:::
想不出障碍因素的时候，请思考下面这些问题。
- 有没有什么想法会妨碍你完成任务？
- 有没有什么行为会妨碍你完成任务？
- 有没有什么习惯会妨碍你完成任务？
- 有没有什么惯性思维会妨碍你完成任务？
- 有没有什么情绪会妨碍你完成任务？

#### 克服障碍的对策
|             因素             |              对策              |
| :--------------------------: | :----------------------------: |
| 因为手机的消息提醒而心不在焉 |  把手机的消息提醒全部关掉障碍  |
|     不管怎样都提不起干劲     |        先工作5分钟障碍         |
|         偷懒不去运动         | 规定“偷懒的话，要向朋友交罚款” |

很多时候，野兽会因为受到挫折而失去干劲。为了保持野兽的动力，有必要事先思考可能会遇到的问题。

#### 提问
把任务变成“提问”形式，是基于一种叫作问题-行为效应,疑问句比陈述句更有影响力.

对于在第1步中选择的每日任务，分别设定相应的“提问”。把每日任务的内容转换成以下格式。

> [自己的名字]在[时间]在[地点]执行[每日任务]吗？

::: tip 案例
每日任务 修改企划书

提问 山田太郎上午9点在公司坐在自己的座位上修改企划书吗？

每日任务 跑2千米

提问 铃木一郎下午7点在健身房跑2千米吗？
:::
　
#### 现实想象
尽可能真实地在脑海中描绘出完成 `提问` 任务之前的过程。

::: tip 案例
提问 山田太郎上午9点在公司坐在自己的座位上修改企划书吗？

现实想象 我在家门口穿跑步鞋 → 我经过附近的便利店去附近的健身房 → 我在健身房，将跑步机设定时速为6千米，跑20分钟

提问 铃木一郎下午7点在健身房跑2千米吗？

现实想象 我将必要的资料整理成一个文件夹 → 我打开平时用的编辑器调整资料的内容格式 → 我粗略浏览资料概要 → 我修改企划书
:::

#### 固定式视觉提示
把实践记录表放在显眼的地方。

野兽只对眼前的事物有反应，如果不能让它定期地想起“提问”的内容，它很快就会忘记。即使在工作的时候走神了，看到“提问”的内容后，野兽也能很容易地重新专注于工作。

同时，你也可以给提示加入视觉要素（所喜欢的风景照片或者卡通人物）
