# 验光

验光单数据

球镜( 近/远视度数 ) 柱镜( 散光 ) 轴位( 散光方向 ) 视力( VA ) 看物体清晰度

验光地点可以选择正规眼科医院和大品牌的连锁眼镜店。

# 镜片

原则: 在经济允许的情况下，选好的。材质: 玻璃，非球面，防蓝光，折射率1.6，阿贝数高的。

  1. 镜片材质
材质 重量 易碎 耐磨
玻璃 重 不易碎 超耐磨
树脂 中 不易碎 耐磨
PC( 塑料 ) 轻 不易碎 不耐磨
  1. 球面与非球面
球面 非球面
成像扭曲, 眼睛易疲劳 边缘不扭曲, 失真度小

  1. 折射率 折射率越高，镜片就越薄，价格越贵。如果近视300以下就少花点钱买个1.6的就行。

  2. 阿贝数( 色散系数 ) 阿贝数越高成像越清晰。抖音logo就是低阿贝数成像的结果。
