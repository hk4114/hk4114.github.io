数据结构与算法： 一组数据的存储结构，算法就是操作数据的方法

数据结构是为算法服务，算法作用在特定数据结构上
比如：因为数组具有随机访问的特点。常用的二分查找算法需要用数组存储数据。如果我们选择链表这种数据结构，二分查找就无法工作了。因为链表不支持随机访问。

### 带着问题学习
1. 算法/数据结构的来历
2. 自身的特点
3. 适合解决的问题
4. 实际的应用场景

1. 边学边练，适度刷题。每周花1-2h集中把涉及的数据结构和算法全都自己写出来。代码实现一遍。
2. 每节课后的思考题
3. 每节课后都写一篇学习笔记或心得
