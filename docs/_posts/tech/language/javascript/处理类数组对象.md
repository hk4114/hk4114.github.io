### 类数组对象是什么？
格式与数组类似，且具有length属性，但是不能使用数组的方法。

比如: `arguments` `NodeList` `HTMLcollection` 等。
```js
const list = {
    '1': 'hua',
    '2': 'logger',
    '3': 'kane',
    length: 3
}
```

### 处理方法

1. Array.from(list)
2. Array.prototype.slice.call(list)
2. Array.prototype.concat.apply([],list)
3. [...list]
