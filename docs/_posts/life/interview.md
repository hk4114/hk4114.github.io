技术工作者的一份简历模板

<!-- more -->

::: tip 提示
内容尽量控制在一页以内
:::

### header 基本信息

```html
<header>
  <!-- 基本信息 -->
  <h4>姓名: </h4>
  <label> 住址:  </label>
  <label> 手机:  </label>
  <label> 邮箱:  </label>
  <label> github:  </label>
  <label> blog:  </label>
</header>
```


### section 1 教育信息

```html
<section>
  <!-- 教育信息 -->
  <label> 大学: start-time to end-time 学士</label>
  <label> 研究生: start-time to end-time 硕士</label>
</section>
```


### section 2 项目经验
> 可以套用以下句式：

```html
<section>
  <!-- 项目经验 -->
  <dl>
      <dt> 项目名称 </dt>
      <!-- 使用下列表述语 将技术核心和个人角色表述清楚  -->
      <dd> 通过_实现了_的效果，提高或改善了(衡量标准)_ </dd>
      <dd> 通过_降低了_至_ </dd>
      <dd> 为了_ 重新设计了_ </dd>
      <dd> 为了_通过_完成了_ </dd>
      <dd> 为了_通过_成立了_ </dd>
      <dd> 为了_通过_将_集成</dd>
  </dl>
</section>
```


### section 3 个人项目

```html
<section>
  <!-- 个人项目 -->
  <dl>
      <dt> 项目名称 </dt>
      <dd> 项目地址 </dd>
      <dd> 项目描述 </dd>
  </dl>
</section>
```


### section 4 技能简介

> os: 根据需要可以和section 1 互换 ( 如果教育信息不够好看的话 :( 哭唧唧

```html
<section>
  <!-- 技能简介 -->
  <ul>
      <li> 精通-非常熟悉，使用顺手，可以立刻进行面试 </li>
      <li> 熟悉-之前用过，可以短时间内熟悉起来。 </li>
  </ul>
</section>
```
