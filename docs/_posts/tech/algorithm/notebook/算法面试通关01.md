### 如何计算算法的复杂度

#### 时间复杂度

- O(1) 常数复杂度 
- O(log n) 对数复杂度
- O(n) 线性时间复杂度
- O(n log n) 线性时间复杂度- 
- O(n^2) 平方复杂度
- O(n^3) 立方复杂度
- O(2^n) 指数复杂度
- O(n!) 阶乘

好几块代码合在一起时，只计算复杂度最高的算法
```js
// O(1) 常数复杂度 
var n = 100;
console.log(n)

// O(n) 线性时间复杂度
for(var i=0;i<n;i++){
    console.log(n)
}

//  O(n^2) 平方复杂度
for(var i=0;i<n;i++){
    for(var j=1;j<=n;j++){
        console.log(n)
    }
}

// O(log n) 对数复杂度
for(var i=1;i<n;i*=2){
    console.log(n)
}

// O(k^n) 指数复杂度
for(var i=0;i<Math.pow(2,n);i++){
    console.log(n)
}

// O(n!) 阶乘
for(var i=0;i<n;i++){
    console.log(n)
}

// O(n^3) 立方复杂度
```
