## 创建github页面

### 物料准备
1. github 账号
2. 创建一个新的仓库(repository)

### 步骤
1. 根目录下创建 `index.html` 文件
2. 点击 settings 
3. GitHub Pages -> Source -> 选择 Branch: main -> (root) -> 点击 `save`
4. 地址栏输入 `<github username>.github.io/<repository name>`
