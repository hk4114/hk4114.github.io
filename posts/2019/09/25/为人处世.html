# 基础

# 【心态】待人真诚

人各有性格，如果做不到包容，至少表达尊重。善意是社交资本中大型通用货币。

具体案例：

  • 对待基层一线多关心体恤
  • 对待社会大哥就当自己是学生
  • 对待领导可以洒脱放开些

# 【心态】金钱观

钱很重要。金钱就像疫苗，它可以避免很多痛苦，但不一定会让你快乐。

用上天赋予自己的实力换来米盐等物资，绝不是贪图财富的恶业。如果因为这件事太低俗而轻蔑它，那就大错特错了，那是不知世事的纨绔子弟才会说的话，太自以为是了。

人当然不能过分追求金钱，金钱是通往幸福的桥梁，但我们不是住在桥上。

再怎么有钱也要居安思危，不要失去了自己生活的能力，这才显出自己的自强自立。 ​​​

# 【心态】谦虚

保持科学严谨的态度。

核心是：解放思想，实事求是。具体问题，具体分析，别拿着锤子看什么都像钉子。

傲慢是对成功者的诅咒（傲慢，坚信自己是对的）。因为成功者常常自己也搞不清楚成功的原因是什么。

当我们用一个思路在一件事上取得成功后，常常会放弃思考，草率地认为这个思路对其他事情也会有效。

但实际上这个思路可能只对特定的事情有效，甚至可能只在特定时间和环境下才对特定的事情有效。可我们会认为自己找到了一条万能的成功之路，直到遭遇失败。

如果诅咒非常强烈，还会让我们即使在遭遇失败后仍然坚信眼前的失败只是成功路上的偶然。

诚恳地相信你也许并不知道最好的解决方法是什么。

大多数人做出糟糕的决策是因为他们过于确信自己是对的，从而无法让自己看到确实存在的更好的选项。

# 【心态】冷静耐心稳定

学会控制自己的情绪。不能让感情代替自己的思想。

怎么做到？

拿笔写出来，问自己得失。

# 【方法论】交朋友

朋友越多越好，敌人越少越好。

所谓团结，就是团结跟自己意见分歧的，看不起自己的，不尊重自己的，跟自己闹过别扭的，跟自己作过斗争的，自己在他面前吃过亏的那一部分人。

通过交流结识更多的人，最好是不同背景的人，并保持联系。

有什么好处吗？未来我们获得一份工作不是因为我们知道什么，而是我们认识谁。

# 如何认识一个人

  • “世上决没有无缘无故的爱，也没有无缘无故的恨”——我们引用这句话的时候，常常偏重前半句，实际上后半句也很有意义。如果一个人和你没有过交往，而初次打交道就很不友善，那要么对方性格如此，要么你就应该思考一下那些既认识你又认识他的人。

  • 看一个人讲不讲理，有一个量化指标：看他说到第几句话时会忘记自己的第一句话。从我的经验来看，一半以上的人在说第二句话时就会忘记自己的第一句话。

  • 不能看他说什么，要看他做什么。表态是廉价的，可行为却不会骗人。

  • 判断人的认知高低有两个方法，第一就是看他怎么面对批评。如果一听到批评就找理由、借口，认知不会太高。第二就是怎么对待任务，如果一安排任务就说困难，就说无法完成，他的认知也不会太高。一个人的成长，归根结底是认知的成长。如果一个人习惯了批评后找自己的问题，他就会成长，如果一个人接受了任务，首先想得是我怎么排除万难完成使命，他一定会成长。

夸人

针对不同年龄段的人群

  • 人年少时没什么东西，便爱听人夸自己；
  • 年稍长，有了这个那个，身体也过了巅峰期，便爱听人夸自己的物件，毕竟使用时间和努力换来的，所以中年恋物慕名，比如资产成就，以至于养的植物猫狗，都是生命伸出的枝节；
  • 老来身体不好，便又爱听人夸自己，因为万物不监牢，生命的安全感便格外要抚慰了。

怎么夸？

首先，夸人一定要是发自内心的，当然还需要一定的技巧：FFC 赞美法则。

先说 Feeling 感受，再说 Fact 事实用来补充细节，最后说 Compare 对比。

举例：

你今天真好看（feeling），穿搭就很时尚，这件 T 恤非常衬你，显得自信开朗（fact），如果是我穿肯定没你好看（compare）。

xx，我特别佩服你（feeling），每次遇到问题向您请教，你不会直接告诉我方案，而是通过提问引导我独立思考（fact）。和你合作的这段时间比以往都进步得快（compare）。

# 进阶

# 克己自律

缺少自律，一切天赋都会随着时间的流逝变得荡然无存，以及造成巨大的伤害。

人类一切苦难的根源是对自己无能的愤怒，而自律是解决人生痛苦的根本途径。

群体的幸福来自于仁慈和公平，个人的幸福来自于节制。---亚当斯密《道德情操论》

对于不可控的事情保持平静与乐观，对于可控的事情保持谨慎与节制。

**利用欲望让自己努力，控制欲望让自己快乐。**能力的提升永远赶不上欲望的膨胀。

以前你爷爷见过的最好看的姑娘就是村口的小芳，村里生活最好的人家也不过顿顿能吃炒鸡蛋。 而现在，你一打开屏幕，满眼的香喷喷紧绷绷鼓胀胀，戴着沛纳海去爱琴海。 你很难不为之所动。然而如果为之所动了，也很难不焦虑。 数字技术让我们获取信息变得异常便利，让我们更容易提升自己的能力，同时也更容易产生欲望。 但能力提升的速度是多项式级别的，而欲望则很容易以指数级速度增长。

所以，想减轻焦虑，提升能力是可选项，控制欲望是必选项。

每年3月5日是学雷锋日，不过现在不怎么提了。雷锋说过一句话：“在工作上，要向积极性最高的同志看齐；在生活上，要向水平最低的同志看齐”。这句话有着深深的五六十年代味道，看起来仿佛已经不合时宜。

其实换成今天的语言就是：提高能力，控制欲望。

不要把成功的装饰误认为成功本身。多看富一代的摸爬滚打，少看富三代的餐桌礼仪，更不要以为餐桌礼仪是发家致富的秘籍。

大名鼎鼎的英国经济学家希克斯谈起意大利文艺复兴（背后是北意大利原生资本主义家族间上百年的明争暗斗）时，说过一句文绉绉的话：人们只会铭记这些果实，但结出果实的时候，秋天已经来了。

# 方法

  • 提高能力，控制欲望。利用欲望让自己努力，控制欲望让自己快乐。

  • 睡觉前，用 5 分钟复盘当天的收获、得到的教训。

  • 看到一篇好文章、和别人聊天后，不要立刻做其他事，也要停下来复盘，用5分钟梳理一下，这么做一年，你会脱胎换骨。

# 别做一个好说话的人（要有原则）

不要为了迁就别人而妥协，结果要比过程重要，如果为了大家感觉舒服而放弃成功，就得不偿失了。

怎么面对贱人，怎么处理贱事是门学问。

面对别人看人下菜的时候，就要敢于撕破脸。

很多刚进社会的年轻人有一个特点，总是对人特别客气，对人很有礼貌。

认为这样能够用友善打动对方，殊不知在对方眼里你这样的就是待宰的肥羊。

社会可不似学校里那样单纯，有些人眼里只有钱，觉得你看起来涉世未深能宰一笔算一笔。

小时候看很多长辈买东西砍价，那语气都要跟骂起来似的，觉得好丢人，殊不知人家社会经验那么多知道对待怎样的人就用怎样的方式。

现在虽不比以前，但很多东西还是没变的。

如果大家进行一些不清楚价格的交易时（例如装修维修甚至理发），态度别放太好，别太客气又买水又买烟的，谈判时别老是先给对方着想，把自己的利益先维护住。真觉得对方做得好，就完事交完钱再感谢人家。

你只要看起来强硬不好欺负，那么对方自然会掂量着点的。

# 做一个靠谱的人

倾向性信任越高，能带给自己较多的交友机会，但是过于信任别人容易被人利用。

保持客观开朗，阳光洒脱。

多和别人交流，但对于深交则必须审慎。

结交有影响力和有能力的人，值得交往的人。

# 提高自己的能力和素养

  • 承认不足并克服它们
  • 找到自己的有点和擅长的工作
  • 通过上课来培养技能

总而言之：勤奋工作，提高自己的能力和素养，为自己赚取更多的收入

# 言出必行

小的错误也会对诚信产生大的影响。

所以需要我们保持诚信，保持稳定，不要不敢承诺或者过分兑现，合理安排时间。

每个人对他人的基本信任度是不同的，有些人天生多疑。如果需要和这类人建立信任关系，就需要我们保持耐心，再不断的交往中保持诚信，演出必行。但也别给自己增加负担，要聪明地工作。

# 提升个人魅力

自信，自信能让人看起来更放松，更游刃有余

  • 避免使用不确定性的语言
  • 细节，比如姿态自然，用适合的音量讲话，与他人进行眼神交流
  • 假装自信直到真的做到
  • 如果真的没把握，交给有把握的人吧 乐观，多称赞别人。有时人们低估内在动机（如认可），高估外在动机（如金钱、休假）。注意过多的赞美会让人觉得虚伪 外表很重要（穿衣、举止得体），可也不要过于依赖以貌取人

# 如何进行讨论和会议

好的协议就是，双方不满意勉强可接受。

冲突并不可怕。冲突对建立良好的人际关系至关重要。

每个人都有自己的原则和价值观，所以每种关系都意味着需要针对人与人如何相处开展一定的谈判或辩论。

你们对彼此的了解，要么让你们关系更密切，要么更疏远。

如果你们的原则一致，你们就能够通过妥协互让解决分歧，你们就会更加亲密。

否则，你们就会分道扬镳。对分歧进行公开讨论能确保消除误解。如果不能持续这样做，观点的分歧就会扩大，直至爆发大的冲突。

搞清楚你是在争论还是在试图理解一个问题，并根据你和对方的可信度，想想哪种做法最合理。

如果双方水平相当，争论就是合理的；如果一方明确比另一方更有知识，就应该让一方做老师，另一方做学生去请教。

要做好这一点，你必须理解“可信度”这一概念。我定义的“可信”的人有两个特征：

  1. 曾反复地在相关问题上有成功的经历（至少成功过三次，拥有过硬履历）
  2. 在被问责的情况下能对自己的观点做出很好的解释。

假如在某个问题上有一个人可信，或者至少比你可信（例如当你和医生讨论你的健康状况的时候），而你和他的观点不同，你就应该清楚地表示你是在提问题，从而试图理解对方的观点。相反，如果你是双方中更可信的人，你就应该礼貌地提醒对方这一点，并建议他向你提问题。

大多数人不愿考虑与他们已经得出的结论不符的信息。当我问其原因时，常见的回答是：“我想自己做决定。”这些人似乎以为，考虑对立观点会威胁他们自行做决定的能力。没有比这更荒谬的了。听听其他人的观点并加以考虑，绝不会削弱你独立思考、自主决策的自由，只会让你在决策时有更宽广的视角。

总而言之：尊重基本事实的前提下，各式各样的偏见并存，远远好过唯一的正确答案。

讨论比静默要好；计算比算计要好；

科学计算和统筹，比“谁哭得声音大”我救谁要好；

明码标价比暗箱操作好；不同群体间进行争夺与博弈，比让领导先走好；

生命有价，总比一些人生命无价，另一些人生命零价好。

# 如果做一个靠谱的职场人

本质是当对别人有用的人。思考怎么给到别人好处、互利互惠。

  1. 件件有着落，事事有回音，凡事有交代。不断证明自己，只有又快又好的完成项目才可能拿到大项目。如果以前没有成功做过某件事，就不要假设自己能做成。 ​​​

  2. 我觉得默认还是要先信任。信任领导、信任下属，信任合作伙伴。信任了才能把事情干起来。而且内部的相互信任也是一种竞争优势。但只要出现一次背叛，就永久拉黑，绝不给第二次机会。

  3. 以结果为导向。

  4. 争取每个机会，四面出击而不是按图索骥。主动寻找机会、抓住机会，越积极越幸运。

  5. 请示工作说方案，布置工作说标准，汇报工作说结果，总结工作说流程，回顾工作说感受
