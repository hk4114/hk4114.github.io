## 饮食

### 原则：
- 多吃对大脑有益的食物。
- 少吃对大脑有害的食物。
- 不限制卡路里。
没有必要减少饭量，吃饱也没关系。
少吃对大脑有害的食物就可以了，不需要完全不吃。

对大脑有益的食物
|   类别   |             例             |                  推荐摄入量                  |    用手测量的分量    |

### 基本设定表

#### 具象想象
1. 不要用专业术语。不要说“建设可持续发展的社会”，而要用“建设混合动力汽车变多的社会”这样的表达。
2. 不要用数字。不要说“一年瘦10公斤”，而要用“能穿上20多岁时买的牛仔裤”这样的表达，后者的效果更好。当然，数字对我们了解自己的完成度来说很重要，但在这一阶段，还是要更加重视目标的具象性。

::: tip 案例
> case 完成企划书　

把企划书交给上司以后松了一口气，并去看了电影

> case每天跑步　

每天跑步增强体魄，总是头脑清醒地完成工作
:::

#### 反向规划
设定在完成最终目标前需要完成的子目标和完成子目标的日期。

需要要注意的是，不是以通常的从现在到未来的方式设定目标，而是从最终目标开始**倒推**，设定短期目标。

::: tip 案例
具象目标: 把企划书交给上司以后松了一口气

反向规划: 在交企划书的前一天把企划书再看一遍→在交企划书的三天前完成企划书→在交企划书的五天前提出解决方案→在交企划书的七天前完成调查
　
目标具象: 每天跑步增强体魄

反向规划: 3个月后累计跑100千米→1个月后累计跑25千米→14天后累计跑10千米
:::

#### 每日任务设定
把目标进行细化是任务管理的基本原则。对将目标细化到什么程度没有固定的标准，以**能在几分钟到一个小时内完成这个目标**为宜。

::: tip 案例
子目标: 在截止日期七天前完成调查和分析

每日任务: 对专业人士进行访谈，从网上收集必要的资料，阅读收集的资料等

子目标: 14天后累计跑10千米

每日任务: 在跑步机上跑1千米，沿着平时的跑步路线跑2千米等
:::

基本设定表 并不是填一次就可以了，我们需要根据项目的情况定期修改。

其中修改最多的是“每日任务设定”。

很多人不能集中注意力是因为在这一步没有合理地细化任务。

当出现这种情况的时候，先回到“每日任务设定”这一步，对任务进行再一次细化。

如果“写企划概要”的任务不能让你集中注意力，就把它分解成2～3个子任务，如“从资料中提取有用的信息”和“整理提取的信息”。

### 实践设定表

#### 每日任务选择
从基本设定表中的每日任务中，选出3～5个“当天必须着手做的事情”或“最长也只需要2～3小时就能完成的事情”。

驯兽师无法同时处理3个以上的信息。总之，一天最多给自己设定5个任务，如果还有多余的时间再追加。

#### 对比目标和障碍  
写出在完成每日任务时可能会发生的问题。

::: tip 案例
如果每日任务是“在跑步机上跑1千米”，至少写出一个会妨碍你完成此任务的因素，如 工作太累，不想跑了,想去看电视等。
:::
想不出障碍因素的时候，请思考下面这些问题。
- 有没有什么想法会妨碍你完成任务？
- 有没有什么行为会妨碍你完成任务？
- 有没有什么习惯会妨碍你完成任务？
- 有没有什么惯性思维会妨碍你完成任务？
- 有没有什么情绪会妨碍你完成任务？

#### 克服障碍的对策
|             因素             |              对策              |
| :--------------------------: | :----------------------------: |
| 因为手机的消息提醒而心不在焉 |  把手机的消息提醒全部关掉障碍  |
|     不管怎样都提不起干劲     |        先工作5分钟障碍         |
|         偷懒不去运动         | 规定“偷懒的话，要向朋友交罚款” |

很多时候，野兽会因为受到挫折而失去干劲。为了保持野兽的动力，有必要事先思考可能会遇到的问题。

#### 提问
把任务变成“提问”形式，是基于一种叫作问题-行为效应,疑问句比陈述句更有影响力.

对于在第1步中选择的每日任务，分别设定相应的“提问”。把每日任务的内容转换成以下格式。

> [自己的名字]在[时间]在[地点]执行[每日任务]吗？

::: tip 案例
每日任务 修改企划书

提问 山田太郎上午9点在公司坐在自己的座位上修改企划书吗？

每日任务 跑2千米

提问 铃木一郎下午7点在健身房跑2千米吗？
:::
　
#### 现实想象
尽可能真实地在脑海中描绘出完成 `提问` 任务之前的过程。

::: tip 案例
提问 山田太郎上午9点在公司坐在自己的座位上修改企划书吗？

现实想象 我在家门口穿跑步鞋 → 我经过附近的便利店去附近的健身房 → 我在健身房，将跑步机设定时速为6千米，跑20分钟

提问 铃木一郎下午7点在健身房跑2千米吗？

现实想象 我将必要的资料整理成一个文件夹 → 我打开平时用的编辑器调整资料的内容格式 → 我粗略浏览资料概要 → 我修改企划书
:::

#### 固定式视觉提示
把实践记录表放在显眼的地方。

野兽只对眼前的事物有反应，如果不能让它定期地想起“提问”的内容，它很快就会忘记。即使在工作的时候走神了，看到“提问”的内容后，野兽也能很容易地重新专注于工作。

同时，你也可以给提示加入视觉要素（所喜欢的风景照片或者卡通人物）

## 创造仪式
> tip 吃饭前可以做个仪式，吃饭时专心吃饭，有利于减肥。

创造仪式时最重要的是以下两点。
1. 决定“做了这个动作以后就要做重要的工作了”。
2. 多次重复固定的步骤。
不应该只是重复无意义的步骤，而应该把对现实有用的动作编入到仪式中。这样能更大限度地激发野兽的动力，最终提高专注力。

### 创造仪式的策略
#### 从简单的任务做起
> 任务完成偏差: 人们没有将注意力集中在耗时长且重要的任务上，而是集中在耗时短且重要性低的任务上。本来应该学习的，却不知不觉地去回复不必要的邮件，这种现象可能出现在任何人身上。

利用任务完成偏差提高专注力，就是从简单地任务做起。

从基本设定表的“每日任务设定”中选出马上就能完成的工作。在一天的开始做这些工作也不错。

仅仅这样，任务完成偏差就能作为野兽力量的助推器发挥作用。

#### 坚持记录
记录的内容和形式不限，记日记、写博客、记账、记录体重变化等都可以。

定期记录一些数据的行为能作为提高专注力的**仪式**发挥作用。

记账越仔细，人主观压力越小，平时工作也更专注。

像记录这样简单的任务就能给野兽带来成就感，提高其专注力。

更重要的原因是自我效能感的提升。坚持记录能让你越来越相信自己是个能干的人。

记录的内容最好是与有助于提高专注力的方法相关的数据。

第1章中提到的MIND记分板和第2章的回报感觉计划等，都是记录训练的好选择。记录自己的进展情况也是非常好的选择。

**记录自己的进展情况时，请注意以下要点。**
1. 想改变行动的时候，只记录自己采取的行动。
2. 想得出结果的时候，只记录得出结果的过程。

如果你的目标是减肥，就记录体重。
如果你想改变饮食习惯，记录下吃过的食物效果会更好。
如果你以减肥为目标，却记录了自己吃过的食物，减肥失败的概率就会变高。
如果你想增加存款，就只记录存款的增减。
如果你想戒烟，就只记录不吸烟的天数。
如果你想坚持运动，就只记录去健身房的天数。
如果弄错，记录的效果就会骤减。记录的时候，请一定将行动与结果相对应。

**记录的效果显现至少需要2个月。**
因为记录不是一种立竿见影的技巧，所以即使几天内感受不到效果，也请不要放弃，请继续坚持记录。

### 小小的不快会让内心变得强大
小小的不快 -> 给你的身心带来轻微压力的事情
- 面对喜欢的酒时保持克制。
- 用平常不习惯用的那只手使用鼠标
- 发觉自己驼背的时候把背挺直。

选择什么样的小小的不快因人而异。忍住不吃夜宵/买了新东西就要扔点旧东西, 什么都可以.
选一些对你来说稍微需要忍耐力的行为。最重要的就是难度的设定。
比如，对于“忍住不吃夜宵”，可能有人怎么都忍不住，有人很轻松就能做到。
如果设定错了“不快”的难度，可能会适得其反。
很多数据显示，最合适的“不快”难度系数为0.8～0.9（即成功率为80%～90%）。
如果高于这个难度，你的心情就会低落，如果你一直成功，就起不到训练野兽的作用。
请选择大部分情况下能忍受、很少会失败的“不快”。

#### 巧用 5的法则
如果找不到难度合适的“不快”，可以试一下“5的法则”。

- 想停下工作休息的时候，再坚持5分钟。
- 想看手机的时候，再工作5分钟。
- 想停止腹肌锻炼的时候，再做5次锻炼动作。
- 无法集中注意力读书的时候，再看5页。

5的法则 能使我们集中注意力，克服 想停止工作、想看手机 这样的小小的不快的体验，会让野兽更自信。
天生就不擅长开始工作的人，怎么都无法开始工作。
遇到这种情况，请像下面这样使用“5的法则”。

情况　
必须做文件了，却无法开始工作对策　
在脑海里倒数“5、4、3、2、1”，数到1的时候开始打字

情况　
想运动却没有干劲对策　
在脑海里倒数“5、4、3、2、1”，数到1的时候开始下蹲

不想工作的时候，倒数“5、4、3、2、1”，数到1的时候开始打字。
虽然这样不一定能让你开始工作，但是总比不采取任何对策只是拖延要好。

**建议大家在使用“5的法则”时，把提示放在能经常看到的地方。**

在纸条上写下
想停下工作休息的时候，再坚持5分钟
无法开始工作的时候，在脑海中进行5秒钟倒计时
等，并把提示纸条放在能经常看到的地方。

### 用仪式叠加引导野兽
不管什么样的仪式，一定要长期坚持下去，多次重复仪式会让野兽认识到“这个行为很重要”。
让仪式自动化，就是把仪式“安装”到野兽体内的过程。
创造出一个仪式后，请尝试仪式叠加。

这是斯坦福大学的行为科学实验室推荐的方法，是将不同的仪式叠加起来的技巧。

例：
仪式1　
每天早上8点，从每日任务中选一个最简单的任务优先处理
仪式2　
完成简单的任务后，开始做难的工作
仪式3　
完成难的工作后，出去跑步10分钟
例：
仪式1　
吃完晚饭后，冥想15分钟
仪式2

### 一周四次，坚持两个月就能使仪式自动化
在仪式完全自动化之前，需要重复这个仪式多少次？
一周去健身房4次以上的人，很可能会坚持锻炼。
一周去健身房少于4次的人，坚持锻炼的可能性大幅降低。

想要某仪式自动化，至少需要一周重复4次该仪式。
如果在第6周之前做不到一周重复4次以上，之后就会恢复原状。

难易程度不同，仪式自动化的时间也不同。早上喝水简单2周搞定，每天50次俯卧撑要253天


## 改写自我形象
当我们无法专注于某事的时候，就会产生to be or not to be，两种状态。只有认清并坚持自己的身份，才能重新集中注意力，消除认知失调的分裂的感觉，

重新定义自我形象，根据目标构建身份，提升专注力。例：如果集中注意力跑步，就把自己定义为跑步运动员。

### 改写自我形象的五种方法
仪式是有效的方法，但是见效时间长

1. 类属性思维。只要热象更优秀人物的形象，我们就很容易更加积极的行动。(看人物传记？)在开始工作前想象对应优秀人物的样子
2. 加头街。不是喜欢什么头衔给自己加什么头衔，而是符合实际情况。比如：第一章的MIND饮食者，第二章的回报感觉实践者（前端技术专家/全栈工程师），确定新头衔后，在开始工作前，再次告诉自己。我是_________.
3. 指导型自我对话(自言自语)不断重复

注意：不要用“我没问题”“今天状态很好”这类激励的对话

::: tip
例
学习时 刚刚停止学习是因为哪里不明白吗?试着想一下其他解决问题的方法，如果想不明白就进入下一题
工作时 为了轻松完成这个文件，能不能找到其他可用资源?努力从已有的信息中提取重要的内容
:::

使用指导型自我对话的时候，关键是把应该做的事情用语言表达出来

::: tip
例
这个问题的重点是什么？
解题的步骤是什么？
:::

觉得自己专注力下降的时候，像“再坚持五分钟”“再做最后一道题”这样自我激励的对话才是有效的


如果想不出好的自我对话，尝试问自己下列问题
- 为什么专注力下降了？因为任务太难了？还是被什么东西干扰了？
- 有没有解决办法
- 享受眼前的任务吗？如果不享受，是什么原因呢？
- 是否接触了大量信息而感到混乱？有没有从大量的信息中提取重要信息的方法呢？
- 为了完成眼前的任务，能否使用其他资源？为了得到有用的资源，我能做什么？
- 这个任务最大的难点是什么？有没有办法解决这个难点
- 最难理解的地方是什么
- 对于无法集中注意力的问题？是否需要向其他人申请帮助？
- 知道自己为什么烦恼吗？如果不知道，怎么样才能知道
- 现在的任务难度适合了？是太难还是太简单了？
- 有没有能让自己感兴趣的方法呢

每次发生问题，就下达明确指令(第一章 克服障碍和对策)

## 观照自我---借助正念找回平静和专注(headspace)

2. 意志力与糖分无关

自我损耗理论说得是情绪控制问题，如何控制拒绝完成特定任务的野兽，要想保持专注，必须控制情绪（比如巧用5的法则）

### 环境管理

#### 整理工作环境
创造仪式的技巧 每天早上开始工作前整理工作环境

#### 准备专用空间
这项任务必须在这个空间里进行。怎么都无法集中注意力的时候就暂时离开这个地方去别处休息

#### 个性化管理空间
开放式工作区比格子间注意力分散的概率高64%，
1. 用隔板或家具分隔功能区（工作/学习/休息）
2. 对自己的办公桌个性化
3. 不在自己工位上吃饭打盹

#### 电子产品管理
1. 手机与工作无关的app单独放一个文件夹，文件夹尽量放在靠后的页面
2. 工作时把画面调成单色，苹果手机设置灰色

#### 声音管理
障噪耳机或者使用音乐

使用音乐注意点：
1. 外向型性格听音乐，内向型性格挺白噪音或氛围音乐
2. 严禁有歌词的曲子
3. 在工作的间隙听音乐

### 克服冲动
1. 情绪评分。在做重要工作的过程中走神的时候，用百分比来记录内心的变化。
比如 在工作中想看与工作无关的网站
无聊向上网，无聊强度40%，还有点焦虑，焦虑20%。仔细想想还有点逃避工作，这种感觉10%。想到这里焦虑减少到了10%
分心的时候就给自己情绪评分，客观观察并记录自己情绪的变化。
2. 情绪的物质化
情绪评分有可能出现一时无法用语言描述自己的情况。这时问自己“如果把现在的情绪比喻成某种物质，那会是什么？”
想象不出来的时候
- 这种情绪是什么颜色
- 这种情绪的大小是什么样的？黄豆/网球/还是高楼
- 这种情绪位于身体的哪个部分，占了多少空间
- 如果触碰这种情绪，是什么样的感觉？是硬/软？光滑/粗糙？
- 这种情绪有温度吗？冷/热/常温？
- 这种情绪有什么动作吗？振动/静止/跳动？如果在动，频率是多少

这时候不要控制自己的情绪。如果感到无聊想停止学习，就给这种情绪评分。然后淡定的继续学习。

## 学会放弃和休息
常见的误解是把自我接纳理解为 不要在意失败 和 做真实的自己
自我接纳应该是在承认自己不完美的前提下冷静分析为何失败，这是达成目标的基础。

### 锻炼自我接纳
#### 自我想象
因失败闷闷不乐，或被消极想法所因扰不能好好集中注意力的时候

步骤：
1. 想象自己因失败而烦恼时，朋友理解和体谅自己并给自己提出建议。
2. 仔细想象朋友说了什么话，将这些话写在纸条上

#### 解离
当自责时，问自己问题

目前糟糕的状况能证明“我没有专注力”“感觉没法完成目标”等想法是正确的吗？

#### 两分钟承诺
> 可以作为第五章所提到的锻炼观照自我能力的方法使用

1. 设定一天中6个必须自我接纳的时间段（起床后或睡觉前等，每次2分钟）。
2. 在这些时间段里回想过去的失败经历和消极的想法，在这2分钟内不对这些经历和想法做任何评判，任其来去。

#### 积极资源法
想更加积极面对失败的人。因不能达成目标而感到自责的时候，尝试一些对工作和学习有帮助的新事物。
1. 选择能发挥自己优势的任务  VIA SMART -> VIA测试了解自己
2. 选择能学到东西的任务

### 休息
1. 微休息 休息40s 看有花草的自然风光图片
2. 任务休息 在进行重要工作之前，先列出几个简单的任务，在做重要且困难的工作间隙，完成简单的任务，比如：看邮件/制定日程表。不需要深入思考马上就能完成的工作
3. 积极休息 轻松散步10min
4. 超积极休息 休息时做比散步更剧烈的运动，睡眠不足或身体极度疲劳严禁期烈运动。
5. 美军快速睡眠法
   1. 前五步和瑜伽大休息式没两样
   2. 反复对自己说“不要思考，不要思考”，重复大约十秒钟。
   3. 想象有一个平静的湖面，你正躺在湖面上的独木舟上，头顶是湛蓝的天空。
   4. 想象你在黑暗的房间里，你躺在吊床上，轻轻摇晃。
